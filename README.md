# JavaBMI_Cal

A Body Mass Index (BMI) calculator available as both a **web interface** and a **Java Swing desktop app**.

---

## 🌐 Web Interface (runs on any device)

Open **`index.html`** in any web browser — no installation needed.  
Works on desktop, tablet, and mobile.

**To use from GitHub Pages** (if enabled):  
`https://<your-username>.github.io/JavaBMI_Cal/`

**To use locally:**  
Download or clone the repo, then double-click `index.html`.

---

## ☕ Java Desktop App

Requires Java 8 or later.

```bash
# Compile
javac *.java

# Run
java Main
```

---

## BMI Categories

| BMI Range | Category |
|-----------|----------|
| < 18.5    | Underweight |
| 18.5 – 24.9 | Normal |
| 25.0 – 29.9 | Overweight |
| ≥ 30.0    | Obese |
